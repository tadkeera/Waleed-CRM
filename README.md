# PharmaComm CRM

تطبيق إدارة علاقات عملاء احترافي لشركات الأدوية والمستلزمات الطبية.

## المميزات الرئيسية

### ✅ إدارة العملاء
- استيراد جهات الاتصال من الهاتف
- إضافة وتعديل العملاء مع عدة أرقام
- تصنيف (طبيب / صيدلي / مدير مشتريات)
- تحديد الكلاس (A/B/C)
- حقول ديناميكية للتخصص والمنطقة مع حفظ تلقائي (Autocomplete)

### ✅ الرسائل الجماعية عبر واتساب
- فلترة متقدمة (تخصص + منطقة + كلاس)
- إرسال رسائل مخصصة باستخدام `{name}`
- آلية متسلسلة آمنة ضد الحظر (تأخير عشوائي 10-15 ثانية)
- فتح واتساب تلقائي لكل عميل

### ✅ لوحة التحكم التحليلية
- إجمالي العملاء + المكتملين
- تنبيهات الانقطاع حسب الكلاس
- إحصائيات التواصل الأسبوعي

### ✅ تتبع المكالمات الفائتة
- نافذة منبثقة فورية عند مكالمة فائتة
- حفظ العميل مباشرة من النافذة

## التقنيات المستخدمة

- **Kotlin** + **Jetpack Compose**
- **MVVM + Clean Architecture**
- **Room Database**
- **Coroutines + Flow**
- **Navigation Compose**
- **WorkManager** (للعمليات الخلفية)
- **GitHub Actions** CI/CD

## المتطلبات

- Android Studio Hedgehog أو أحدث
- JDK 17
- Android SDK 34

## كيفية البناء والتشغيل

```bash
./gradlew assembleDebug
```

الملف الناتج: `app/build/outputs/apk/debug/app-debug.apk`

## CI/CD (GitHub Actions)

عند كل `push` إلى `master`:

- بناء تلقائي
- رفع ملف APK كـ **Artifact**
- إنشاء **Release** جديد مع الملف

## الصلاحيات المطلوبة

- `READ_CONTACTS`
- `READ_PHONE_STATE`
- `READ_CALL_LOG`
- `SYSTEM_ALERT_WINDOW` (للـ overlay)
- `INTERNET`
