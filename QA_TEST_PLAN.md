# خطة الجودة والاختبار - Waleed CRM v2.4

## اختبارات آلية مضافة
- اختبار تنسيق رقم الهاتف اليمني.
- اختبار توحيد السابقة `د.` وعدم تكرارها.
- اختبار البحث العربي وتطبيع الحروف.
- اختبار حفظ اسم الطبيب بدون السابقة داخل قاعدة البيانات.

`app/src/test/java/com/waleed/crm/data/ModelsUtilityTest.kt`

## قائمة فحص يدوية قبل النشر
- تثبيت التطبيق كتحديث فوق v2.1 أو أحدث بدون حذف.
- فتح صفحات: الأسماء، الأطباء، المعرض، الداشبورد، استيراد/تصدير.
- إضافة طبيب جديد وتعديل بياناته وحذفه.
- تجربة البحث باسم الطبيب بعد 3 أحرف.
- تصدير CSV للعملاء وسجل الرسائل والحملات.
- إنشاء نسخة احتياطية مشفرة.
- إرسال واتساب نص فقط ومرفق فقط ونص مع مرفق.
- التأكد من عدم طلب صلاحيات غير ضرورية عند التشغيل.
