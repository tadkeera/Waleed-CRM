# Waleed CRM v2.5 - UI/UX + Follow-ups

## المرحلة العاشرة
- إضافة كرت مختصر للمتابعات داخل الداشبورد.
- تحسين شاشة تفاصيل العميل بتنظيم أقسام البيانات والمتابعات والسجل.
- إضافة شاشة مستقلة للمتابعات في شريط التنقل السفلي.

## المرحلة الحادية عشرة
- إضافة جدول follow_ups في قاعدة البيانات.
- إنشاء متابعة لطبيب من صفحة التفاصيل: غداً، بعد أسبوع، بعد شهر.
- عرض المتابعات المعلقة مع تمييز المتأخر.
- إنهاء أو حذف المتابعة.
- فتح تفاصيل الطبيب أو واتساب من المتابعة.
