# دليل الاستيراد والتصدير والنسخ الاحتياطي - Waleed CRM

إصدار 2.3 يضيف شاشة "استيراد/تصدير" داخل التطبيق.

## التصدير
- تصدير العملاء والأطباء بصيغة CSV.
- تصدير سجل الرسائل بصيغة CSV.
- تصدير تقرير حملات واتساب بصيغة CSV.

## الاستيراد
يدعم استيراد CSV بالأعمدة التالية بالترتيب:
`name, phone, type, specialization, class, location, notes`

## النسخ الاحتياطي المشفر
- يتم إنشاء ملف `.wcrm` مشفر باستخدام AES/CBC و PBKDF2.
- يجب حفظ كلمة المرور؛ لا يمكن استعادة النسخة بدونها.
- الملف يشارك عبر Android Sharesheet من cache آمن عبر FileProvider.
