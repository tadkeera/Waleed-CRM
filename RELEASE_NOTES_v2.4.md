# ملاحظات إصدار Waleed CRM v2.4

## المرحلة الثامنة: الجودة والاختبار
- إضافة اختبارات Unit Test لدوال البيانات الأساسية.
- توثيق خطة اختبار QA قبل النشر.

## المرحلة التاسعة: النشر والإصدارات
- رفع الإصدار إلى `versionName = 2.4` و `versionCode = 15`.
- تحديث ملف APK الرئيسي داخل المستودع: `Waleed-CRM.apk`.
- استمرار استخدام مفتاح التوقيع الثابت لضمان قبول التحديثات القادمة بدون حذف التطبيق، بشرط أن تكون النسخة المثبتة v2.1 أو أحدث.
- إضافة ملاحظات إصدار واضحة داخل المستودع.
